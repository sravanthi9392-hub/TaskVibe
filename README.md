# 🚀 TaskVibe - Smart Task Management Platform

A responsive full stack web application built using Python Flask, HTML5, CSS3, JavaScript, Bootstrap 5, and SQLite.  
This project allows users to securely sign up, log in, manage their personal tasks efficiently, filter through data, and view detailed task completion statistics in an attractive SaaS-style dashboard.

## Features
- **Secure Login & Signup:** Protects user accounts with safe, hashed passwords.
- **SaaS Dashboard:** Real-time stats counting total, pending, in-progress, and completed tasks.
- **Progress Tracking:** Interactive progress bar showing task completion percentages instantly.
- **Full CRUD Actions:** Easily add, view, edit, and delete tasks through simple forms.
- **Advanced Filters:** Search tasks by title, status, or priority level instantly.
- **Smart Sorting:** Organize tasks by newest first, oldest first, or upcoming due dates.
- **Safe Deletion:** Dynamic popup alerts to prevent accidental task drops.
- **Session Security:** Private user profiles ensuring you only see your own tasks.
- **Responsive Design:** Premium layout optimized for mobile, tablet, and desktop screens.
